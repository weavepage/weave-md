# Examples

This directory contains non-normative examples demonstrating Weave Markdown features.

These examples are for illustration and learning purposes. For normative behavior, see the `conformance/` directory.

## Example Documents

- [Weave Format Demo](weave-format-example.md)

## Usage

Examples show how to:

- Structure a Weave document
- Use weave format extensions
- Use node links effectively
- Organize sections
- Handle references and cycles
- Export to different formats
